# Skill Swap - Frontend

This folder contains the Vite + React frontend for Skill Swap.

## Setup

1. Copy `.env.example` to `.env` and update `VITE_API_URL` to your backend (default http://localhost:5000).
2. Install dependencies:

   npm install

3. Run dev server:

   npm run dev


Pages:
- /signup
- /login
- /dashboard
- /find
- /video/:roomId
